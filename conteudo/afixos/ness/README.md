# Afixos

## NESS

    O sufixo "-ness" em inglês é usado para formar substantivos a partir de adjetivos, indicando uma qualidade, estado, ou grau. Ele é adicionado ao final de alguns adjetivos para criar substantivos que descrevem a condição ou característica associada ao adjetivo. Por exemplo:

    1. Happiness (Felicidade): de "happy" (feliz).
    2. Darkness (Escuridão): de "dark" (escuro).
    3. Kindness (Bondade): de "kind" (gentil).
    4. Sadness (Tristeza): de "sad" (triste).


### a grandeza = the bigness
    big = grande
    bigness = grandeza

### a frieza = the coldness
    cold = frio
    coldness = a frieza

   
### a impotência = the powerlessness
    power = poder
    powerless = incapaz
    powerlessness = a impotência
